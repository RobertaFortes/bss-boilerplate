# grunt-bss-boilerplate
Boilerplate front-end with Grunt

<br/>
## Plugins 

* Clean
* Concat
* Imagemim
* Jshint
* Sass
* Uglify
* Watch
* Spritesmith
<br/>

## Usage

1. Install packages and initial structure<br/>
`npm install`

2. Init project <br/>
`grunt`

3. Watch project<br/>
`grunt build`
